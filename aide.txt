-- Aide du Retro System --

Bienvenue dans l'interface Retro System. Voici une liste des commandes disponibles :

1. **Calculatrice** :
    - Entrez une équation mathématique dans le champ de calcul et appuyez sur "Calculer".
    - Vous pouvez aussi dire "Calcule [expression]" pour que le robot effectue le calcul à votre place.
    - Exemple de commande vocale : "Calcule 2 plus 2".

2. **Recherche Web** :
    - Tapez une recherche dans le champ et appuyez sur "Rechercher" pour ouvrir la recherche Google dans un nouvel onglet.
    - Vous pouvez aussi dire "Recherche [terme]" pour que le robot effectue la recherche pour vous.
    - Exemple de commande vocale : "Recherche Paris".

3. **Synthèse Vocale** :
    - Tapez un message dans le champ de texte et appuyez sur "Parler" pour que le robot le prononce.
    - Exemple de commande vocale : "Dis bonjour".

4. **Commandes vocales** :
    - Le robot peut exécuter une série de commandes vocales. Voici quelques exemples :
        - "Calcule [expression]" : Effectue un calcul.
        - "Recherche [terme]" : Ouvre une recherche Google pour le terme donné.
        - "Dis [message]" ou "Répète [message]" : Le robot répétera ce que vous avez dit.
        - "Descends" : Fait défiler la page vers le bas.
        - "Monte" : Fait défiler la page vers le haut.
        - "Actualise" : Recharge la page.
        - "Mode sombre" : Active le mode sombre.
        - "Mode clair" : Active le mode clair.
        - "Traduit [texte] en [langue]" : Traduit un texte dans la langue spécifiée (exemples : anglais, espagnol, etc.).
        - "Rose et bleu" : Change le fond en dégradé de rose et bleu.

5. **Commandes supplémentaires** :
    - "Jarvis" : Le robot répond avec "Bonjour monsieur".
    - "Alors" : Le robot répond avec "Oui, j'écoute".
    - "Merci" : Le robot répond avec "De rien".

6. **Mode Sombre / Mode Clair** :
    - Le mode sombre change l'arrière-plan en une couleur plus sombre, pour une meilleure lisibilité dans des environnements peu éclairés.
    - Le mode clair est la version classique avec des couleurs plus claires.

7. **Changer de fond** :
    - Vous pouvez dire "Rose et bleu" pour changer le fond en un dégradé de rose et bleu.

Merci d'utiliser Retro System ! Si vous avez d'autres questions, n'hésitez pas à demander.
