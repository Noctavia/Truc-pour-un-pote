------------------------------------------
Aide - Commandes disponibles
------------------------------------------

🔹 Commandes de base :
- "Calcule [expression]" → Effectue un calcul mathématique (ex : "Calcule 5 + 3")
- "Recherche [terme]" → Recherche un terme sur Google (ex : "Recherche Python")
- "Dis [message]" → Fait parler l'IA avec un message donné (ex : "Dis Bonjour !")
- "Descends" → Descend la page web actuelle.
- "Monte" → Monte la page web actuelle.
- "Actualise" → Actualise la page.
- "Mode sombre" → Active le mode sombre (fond noir).
- "Mode clair" → Active le mode clair (fond blanc).
- "Affiche la date" → Affiche la date et l'heure actuelle.
- "Lance un minuteur de [x] minutes" → Lance un minuteur avec un délai en minutes.
- "Mets une alarme à [hh:mm]" → Mets une alarme à une heure précise.
- "Convertis [nombre] [unité] en [unité]" → Convertit une unité en une autre (ex : "Convertis 10 kilomètres en miles").
- "Quelle est la météo à [ville]" → Affiche la météo de la ville donnée.
- "Prends une note [note]" → Prend une note et la sauvegarde.
- "Raconte-moi une blague" → Raconte une blague aléatoire.
- "Génère un mot de passe" → Génère un mot de passe aléatoire.
- "Compte les mots" → Compte le nombre de mots dans une phrase donnée.

🔹 Commandes système :
- "Ouvre [nom d'un site]" → Ouvre un site web spécifique (ex : "Ouvre YouTube")
- "Ferme l'onglet" → Ferme l'onglet actuel.
- "Éteins l'ordinateur" → Arrête le PC (Windows uniquement, avec une confirmation).
- "Redémarre l'ordinateur" → Redémarre le PC.

🔹 Commandes multimédia & son :
- "Joue [nom de la musique]" → Lance une musique sur YouTube.
- "Mets en pause" → Met en pause la musique en cours.
- "Reprends" → Reprend la musique en cours.
- "Monte le son" → Augmente le volume du PC.
- "Baisse le son" → Diminue le volume du PC.
- "Coupe le son" → Désactive le son du PC.
- "Rétablis le son" → Active le son du PC.

🔹 Commandes d'interaction avancée :
- "Envoie un message à [nom]" → Envoie un message (ex : via WhatsApp Web ou Discord).
- "Lis mes notifications" → Annonce les notifications importantes.
- "Traduis [texte] en [langue]" → Traduit une phrase en une autre langue.

🔹 Commandes utilitaires :
- "Définition de [mot]" → Donne la définition d'un mot (via une API de dictionnaire).
- "Convertis [nombre] [unité] en [unité]" → Convertit des unités (ex: "Convertis 10 kilomètres en miles").
- "Rappelle-moi de [tâche] à [heure]" → Ajoute un rappel automatique.

------------------------------
          Instructions :
------------------------------

- Pour utiliser une commande, tapez-la dans l'interface ou activez la reconnaissance vocale en cliquant sur le bouton "Activer/Désactiver la reconnaissance vocale 🎤".
- Certaines commandes nécessitent une précision dans la saisie, telles que les commandes liées aux heures ou aux conversions de devises.
- Si vous avez besoin d'aide pour une commande spécifique, vous pouvez demander à nouveau la liste des commandes en écrivant "Aide" ou en activant la commande vocale.
